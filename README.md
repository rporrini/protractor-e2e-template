# protractor-e2e-template

Template project for writing e2e tests with [protractor](https://www.protractortest.org/#/) and [mocha](https://mochajs.org/)

### Dependencies

* ```node```
* ```chrome```

### Installation

* ```git clone https://github.com/rporrini/protractor-e2e-template.git```
* ```npm i```

### Run the e2e tests

* ```npm t```
